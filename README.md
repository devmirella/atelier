# Atelier

Projeto autoral em desenvolvimento focado em HTML, CSS e JavaScript puro.

O Atelier é um projeto front-end que une estrutura, lógica e identidade visual, com foco na construção gradual de funcionalidades reais e organização de código.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (vanilla)
- LocalStorage

## Estrutura atual

O projeto conta atualmente com:
- Tela de login e cadastro com validações
- Controle de sessão com LocalStorage
- Página Home com navegação protegida
- Página Minha Arte com sistema de favoritos

## Status

Projeto em desenvolvimento.

Algumas funcionalidades já estão implementadas, enquanto outras páginas e fluxos ainda estão em fase de construção e refinamento.

## Próximos passos

- Criação das páginas Minhas Palavras e Inspiração
- Refinamento visual das telas existentes
- Evolução das interações e fluxos do usuário
